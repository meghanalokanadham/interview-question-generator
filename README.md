# Interview Question Generator

## Overview
This project is a Python-based application that generates interview questions based on input. It is built with a simple modular structure where the backend logic and user interface are separated for better clarity and maintainability.

The project includes:
- app.py → Main backend logic for generating questions
- app_ui.py → User interface layer
- requirements.txt → Required dependencies

## Project Structure
.
├── app.py
├── app_ui.py
├── requirements.txt
└── README.md

## Features
- Generates interview questions dynamically
- Clean separation of backend and UI
- Lightweight and easy to run
- Beginner-friendly Python project structure
- Easily extendable for more features

## Installation

### Clone the repository
git clone <https://github.com/meghanalokanadham/interview-question-generator>
cd <interview-question-generator>

### Install dependencies
pip install -r requirements.txt

## How to Run

Run the main application:
python app.py

If UI is separate:
python app_ui.py
